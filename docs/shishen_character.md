# 十神性格文案与打印协议索引

> **目的**：作为工程师索引与未来维护入口，记录所有十神的「主要性格卡」文案与「性格快速汇总」的一句版库。

---

## 1. 打印协议 (Printing Contract)

### 1.1 主要性格卡（Long-form Talent Cards）

打印位置：`主要性格` 区域

#### 印星（简化版 2 行）

印星只打印 2 行：**性格画像** + **提高方向**，不区分思维/社交/兴趣/生活方式。

3 档选择规则 (`pian_ratio` threshold)：
| 档位 | 条件 | 输出 |
|------|------|------|
| 正印（纯正印/正印主导） | pian_ratio ≤ 0.30 或 偏%=0 | 正印 2 行卡 |
| 正偏印混杂 | 0.30 < pian_ratio ≤ 0.60 | 混杂 2 行卡 |
| 偏印（纯偏印/偏印主导） | pian_ratio > 0.60 或 正%=0 | 偏印 2 行卡 |

#### 财星（简化版 2 行）

财星只打印 2 行：**性格画像** + **提高方向**，不区分思维/社交/兴趣/生活方式（与印星相同格式）。

3 档选择规则 (`pian_ratio` threshold)：
| 档位 | 条件 | 输出 |
|------|------|------|
| 正财（纯正财/正财主导） | pian_ratio ≤ 0.30 或 偏%=0 | 正财 2 行卡 |
| 正偏财混杂 | 0.30 < pian_ratio ≤ 0.60 | 混杂 2 行卡 |
| 偏财（纯偏财/偏财主导） | pian_ratio > 0.60 或 正%=0 | 偏财 2 行卡 |

#### 比劫（简化版 2 行）

比劫只打印 2 行：**性格画像** + **提高方向**，比肩/劫财共用同一套文案，不区分正偏。

#### 官杀（简化版 2 行）

官杀只打印 2 行：**性格画像** + **提高方向**，不区分思维/社交/兴趣/生活方式（与印星/财星/比劫相同格式）。

3 档选择规则 (`pian_ratio` threshold)：
| 档位 | 条件 | 输出 |
|------|------|------|
| 正官（纯正官/正官主导） | pian_ratio ≤ 0.30 或 七杀%=0 | 正官 2 行卡 |
| 官杀混杂 | 0.30 < pian_ratio ≤ 0.60 | 混杂 2 行卡 |
| 七杀（纯七杀/七杀主导） | pian_ratio > 0.60 或 正官%=0 | 七杀 2 行卡 |

#### 食伤（完整版 5 栏）

每张卡结构：
1. **共性行** (Common trait): 1 行，所有档位都打印
2. **主卡** (Main card): 5 行 × 5 栏位（思维天赋/社交天赋/兴趣取向/生活方式/提高方向）
3. **补充句** (Supplement): 可选 1 行，仅在「X主导」档位追加

5 档选择规则 (`pian_ratio` threshold)：
```
pian_ratio = 偏% / (正% + 偏%)
```

| 档位 | 条件 | 输出 |
|------|------|------|
| 纯正 | 偏%=0 | 共性 + 正主卡 |
| 纯偏 | 正%=0 | 共性 + 偏主卡 |
| 正主导 | pian_ratio ≤ 0.30 | 共性 + 正主卡 + 偏补充 |
| 各半 | 0.30 < pian_ratio ≤ 0.60 | 共性 + 融合版 |
| 偏主导 | pian_ratio > 0.60 | 共性 + 偏主卡 + 正补充 |

### 1.2 性格快速汇总（Quick Summary）

打印位置：紧随「其他性格」区域之后

结构：
1. **总览** (Overview): 列出所有主要性格，固定顺序：财 → 印 → 食伤 → 比劫 → 官杀
2. **空行**
3. **思维天赋** (Mind): 一句版（支持：食伤 + **印星性格画像** + **财星汇总** + **比劫汇总** + **官杀汇总**）
4. **社交天赋** (Social): 一句版（支持：食伤，**印星/财星/比劫/官杀不出现**）

> **印星特殊处理**：印星在快速汇总中只出现在 MIND 区块，使用性格画像文案（不区分思维/社交），不再出现在 SOCIAL 区块。

> **财星特殊处理**：财星在快速汇总中只出现在 MIND 区块，使用新版汇总文案（不区分思维/社交），不再出现在 SOCIAL 区块。

> **归一化说明**：比劫在 quick summary 中对 比肩/劫财/比肩劫财 做归一化，三种情况共用同一套一句版文案（key：比肩劫财）。

命名规则：

|----|------|------|------|
| 财 | 正财 | 偏财 | 正偏财 |
| 印 | 正印 | 偏印 | 正偏印 |
| 食伤 | 食神 | 伤官 | 食神伤官 |

## 2. 主要性格卡文案库

### 2.1 印星天赋卡（简化版 2 行）

> **注意**：印星不再使用共性行+5栏结构，改为只打印 **性格画像** + **提高方向** 2 行。

**正印卡（纯正印 / 正印主导）：**
```
- 性格画像：善良仁慈，守规矩、讲原则，给人可靠、有信用、值得信任的感觉；更偏主流与体系化的路子。
- 提高方向：不要过分墨守成规，不要过分善良，太容易相信别人，与人相处注意边界感。
```

**偏印卡（纯偏印 / 偏印主导）：**
```
- 性格画像：偏独处，容易被小众非主流吸引；对他人情绪与状态变化更敏感；在自己真正喜欢的领域，往往有独特的天分与研究劲。
- 提高方向：多做事、少空想；先做后看，用行动校准；在不确定中保持坚定信念与节奏感。
```

**正偏印混杂卡（各半）：**
```
- 性格画像：心思缜密、学习能力强，既守规矩也爱探索；对小众事物兴趣强，做事看意义驱动，脑子里常在「求稳」和「理想乌托邦」之间摇摆。
- 提高方向：多做事、少空想，用行动校准，不要过分善良，与人相处注意边界感。
```

### 2.2 财星天赋卡（简化版 2 行）

> **注意**：财星不再使用共性行+5栏结构，改为只打印 **性格画像** + **提高方向** 2 行（与印星相同格式）。

**正财卡（纯正财 / 正财主导）：**
```
- 性格画像：务实、稳健，勤俭节约，讲信用，工作能力强，做事踏实肯干、一步一步把结果做出来。更喜欢稳定、可预期的收入来源，愿意靠持续投入换长期的确定回报。
- 提高方向：在稳的基础上保留弹性；适当允许享受与试错，不要因为过度谨慎错过窗口，同时别把压力全揽在自己身上。
```

**偏财卡（纯偏财 / 偏财主导）：**
```
- 性格画像：行动力强，有想法往往会立刻付诸行动。社交能力强，出手阔绰，浪漫。更容易对钱、资源、机会、交易这些现实结果上心，也更吃「新鲜感」和反馈。
- 提高方向：防止花钱大手大脚，注意欲望与比较心，防止欲望太多而不知足。
```

**正偏财混杂卡（各半）：**
```
- 性格画像：对钱、资源、机会、交易这些现实结果上心。容易同时保持两份工作，或在主要工作之外有其他的收入。工作能力，社交能力强。既能踏实做事，也擅长跑合作，开辟新的机会和渠道。
- 提高方向：防止欲望太多而不知足，注意不同工作之间的精力调配。
```

### 2.4 官杀天赋卡（简化版 2 行）

> **注意**：官杀不再使用共性行+5栏结构，改为只打印 **性格画像** + **提高方向** 2 行（与印星/财星/比劫相同格式）。

**正官卡（纯正官 / 正官主导）：**
```
- 性格画像：稳重、有条理，领导能力强，守信用，靠谱有底线。更习惯按规矩和流程做事；对「有没有越界、有没有踩线」比较敏感。更喜欢有标准、有评判体系的环境；更认可主流路径与权威规则，重秩序与节奏，做事偏稳。
- 提高方向：在守规矩的同时允许灵活调整，不被主流、社会眼光过分裹挟。
```

**七杀卡（纯七杀 / 七杀主导）：**
```
- 性格画像：反应快、决断力强，遇事能快速做决定并承担后果；抗压能力强，行动导向明显，更习惯用结果说话。自我管控能力强，目标感很强，不达目标不轻易停下。强调效率，精神容易紧绷，但也更能扛住压力。
- 提高方向：学会用合适的方式缓解压力，控制脾气。
```

**官杀混杂卡（各半）：**
```
- 性格画像：自我管控能力强，领导能力强，目标感很强，不达目标不轻易停下。遇事更敢做决定，也更敢承担后果，强调效率。
- 提高方向：精神容易紧绷，适当学会放松，控制脾气；允许犯错与调整，一次失误不代表全盘失败。
```

### 2.5 食伤天赋卡

**共性行：**
```
- 食伤共性：重表达与呈现，喜欢让想法落地成型；对「好不好玩、有没有意思、能不能展示」更敏感，也更在意生活的趣味与质感。
```

**食神主卡 5 栏：**
```
- 思维天赋：轻松自然、注重感受，擅长把事情做得舒服、让过程愉快；对「享受当下」有天然敏感度，做事不太喜欢逼太紧。
- 社交天赋：亲和、好相处，给人放松、没压力的感觉；习惯用温和的方式表达，不爱冲突也不爱争论。
- 兴趣取向：偏好轻松愉快、感官友好的领域；对美食、休闲、娱乐、生活美学容易有兴趣，讨厌紧绷和苦行。
- 生活方式：重舒适与节奏感，追求「过得好」多于「跑得快」；容易满足但也容易懒散，需要外力推一把。
- 提高方向：在舒适区之外找一点挑战；保持享受生活的能力的同时，不让「舒服」变成「不想动」。
```

**伤官主卡 5 栏：**
```
- 思维天赋：创意强、表达欲旺，喜欢打破常规、追求与众不同；习惯质疑权威与既有规则，对「有没有新意」特别敏感。
- 社交天赋：个性鲜明、锋芒外露，给人「有想法、有态度」的印象；敢说敢表达，但容易因直言惹争议。
- 兴趣取向：偏好创意型、表现型、颠覆型的领域；对艺术、写作、技术创新容易有兴趣，讨厌循规蹈矩和无趣。
- 生活方式：强调自我表达与存在感，讨厌被压制或被同质化；容易不服管、不服输，情绪起伏也更明显。
- 提高方向：把锋芒转化成作品；学会在表达观点时兼顾对方感受，把「叛逆」变成「创造」。
```

**融合版 5 栏（食神伤官各半）：**
```
- 思维天赋：既追求轻松愉快，也追求创意表达；「舒服」和「出彩」两种做事风格并存，随场景切换。
- 社交天赋：既亲和好相处，也个性鲜明、敢说敢表达；能用温和方式建立关系，也能用态度赢得关注。
- 兴趣取向：一边喜欢轻松休闲、感官享受，一边对创意与表现有追求；既要「好玩」，也要「有意思」。
- 生活方式：既重舒适与节奏，也强调自我表达；容易在「享受当下」和「想要更多」之间来回。
- 提高方向：用舒适托住表达；在保持生活趣味的同时，把创意落地成可被看见的成果。
```

**补充句：**
```
- 伤官补充：同时带一点表达欲与锋芒，在舒服之余也想「亮一下」，偶尔会有点小叛逆。
- 食神补充：同时带一点轻松与柔和的底色，锋芒之外也懂得享受生活，不会一直绷着。
```

**各半提醒句（食伤各半时追加）：**
```
- 食伤各半提醒：两种能量都想要出口——享受的时候想表达，表达完又想放松；重点是找到一个「既舒服又能展示」的节奏。
```
